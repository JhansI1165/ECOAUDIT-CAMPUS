# ECOAUDIT-CAMPUS
EcoAudit: Strategic Energy Dashboard for GNITS Campus. I built an IoT-driven platform to manage 181kWp of solar power and reduce a ₹3.7L monthly bill. By integrating ESP32 sensors with a clean, interactive UI, the tool automates BEE lighting audits (ILER) and provides a 3-step solution to shave peak demand and avoid grid penalties
